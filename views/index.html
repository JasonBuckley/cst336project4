About Windows

Windows is an operating system created by Microsoft for personal computers that provides the user with an easy to use graphical user interface (gui). It is currently the most used operating system in the realm of personal computers. The operating system started as a basic gui extension to MS-DOS in 1985. Within in a decade Windows and DOS were fully integrated with the release of Windows 95 that offered internet support through its Internet Explorer browser. As technology entered into the 2000s era Windows became a more robust operating system that could handle more powerful code, and provide more customization for the user. Since the release of Windows 8 users are able to customize their preferences on one computer, and log into another Windows machine using the same settings. The final addition to the Windows family, Windows 10 brought the feature of a digital assitance cortana. So over the years Windows has developed into a powerful operating system that is able to support countless programs, and provide a wide range of customization for the user.

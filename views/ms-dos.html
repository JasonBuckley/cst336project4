Microsofts Disk Operating System, MS-DOS for short is a text based operating system that was first seen in IBM computers in 1982. However, Microsoft did not make the operating system from the ground up, but rather bought the rights of DOS from Quick and Dirty Operating Systems, and tweaked it for IBM's needs. It was the dominant operating system of the 1980s, and mid 1990s for personal computers until it was displaced by operating systems using graphical user interfaces (gui). MS-DOS saw its final release in September 2000, and is still avaiable to the public as free software. Some embedded systems still use MS-DOS for its simple architecture, and low processing needs. In addition, Windows continues to have a MS-DOS like look to there command line interfaces.

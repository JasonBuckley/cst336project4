Windows 10

Windows 10 is the successor to the windows 8 operating system, and the last Windows. Microsoft has decided to release continous updated builds rather than providing a new operating system every few years. As well, they have provided more optimization towards touchscreen interfaces to be more inclusive towards 2 in 1 computer and tablets such as their surface pro. Windows also took the step back to a desktop interface rather than continuing with the idea of startup screen after logging in. Currently 60% of active Windows computers are running Windows 10.

Additional Features that Windows 10 Brings to Users:

  • Edge: Microsofts new Internet browser
  • Facial Recognition: can now login with facial recognition
  • Fingerprint: can now login with fingerprint
  • Virtual Desktop: allows the user to open multiple desktop instances
  • Cortana: voice assistant ai
